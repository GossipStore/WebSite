    Commands

    !lock Esse comando pode bloquear um chat (Bloqueia apenas para membros). ADMS / Atendentes / Donos
    !UnLock Esse comando pode desbloquear um chat (Desbloqueia para todos). ADMS / Atendentes / Donos
    !Ticket Esse comando manda um setup ticket, onde os usuários podem cirar tickets de compra. ADMS / Atendentes / Donos
    !Mute @user Esse comando muta o @user por 15 minutos. ADMS / Atendentes / Donos
    !Ban @user Esse comando bane o @user permanentemente (Pode ser desfeito por outro comando). ADMS / Atendentes / Donos
    !Pix 1/2 Esse comando manda o pix 1 ou 2 para que o usuário possa pagar pelo produto. ADMS / Atendentes / Donos
